# Random Recipe Ideas

- Recipe ideas for when you don't know what to eat

## What is this?
- Website that generates random quotes and colors of inspiration.

## How does it work? 
- Predefined dishes and gradients are loaded from a JSON file
- On click of a button, a random dish and gradients are selected and displayed on the page
- DOM manipulation is used to update the content and style of the page with the new dish and gradient

## How to run the project?
- Preview on localhost i.e. Codespases: `npm run dev`
- Build for production and deployment: `npm run build`

## Tech Stack
- Node
- npm
- Vite
- Tailwind CSS
